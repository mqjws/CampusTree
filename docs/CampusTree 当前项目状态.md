# CampusTree 当前项目状态

更新时间：2026-06-24

## 总体判断

CampusTree 当前已经可以作为课程设计的本地 MVP 演示项目使用。

但还不能称为完整的生产级最终版本，因为 AI 功能和真实生产部署还没有完成。

## 当前分支情况

本地当前分支：

```text
frontend-dev
```

本地检测到的远端分支：

```text
origin/main
origin/frontend-dev
```

当前情况：

- `main` 分支包含当前完整 MVP 版本，包括前端页面和前后端联调。
- `frontend-dev` 分支为正在开发未来的版本。
- 如果需要在本地运行完整项目，目前应该使用 `main` 分支。

## 已完成的 MVP 功能

### 2026-06-24 新增

- 访客模式：首页和帖子详情支持未登录浏览公开内容。
- 登录页支持“以访客模式访问”。
- 访客不能发帖、评论、点赞、举报。
- 发帖正文允许为空，取消 10 字以上限制。
- 发帖支持“仅注册用户可见”。
- 用户可在个人中心和本人帖子详情页删除自己发布的帖子。
- 帖子举报功能。
- 管理员后台扩展为用户管理、帖子管理、举报管理、敏感词管理。。
- 管理员可封禁、解封、删除普通用户，并删除任意帖子。
- 被封禁用户不能登录，也不能继续使用受保护接口。

### 后端

- FastAPI 应用结构。
- 通过 SQLModel 连接 PostgreSQL。
- Alembic 数据库迁移。
- 用户注册。
- 用户登录。
- JWT 身份认证。
- 获取当前登录用户。
- 查询当前用户发布的帖子。
- 查询当前用户发表的评论。
- 查询当前用户统计数据。
- 修改密码。
- 帖子创建、列表、详情、修改、删除。
- 评论创建、列表、详情、修改、删除。
- 帖子点赞和取消点赞。
- 帖子举报和举报状态处理。
- 访客与注册用户可见性控制。
- 管理员用户管理、帖子管理、举报管理。
- 默认管理员账号自动确保。
- 业务接口统一成功和错误响应格式。
- 已配置本地前端开发所需的 CORS。

### 前端

- Vue 3 + Vite + TypeScript 项目结构。
- Element Plus UI 组件库集成。
- Pinia 状态管理。
- Axios 请求封装，并自动携带 token。
- Vue Router 路由和登录守卫。
- 登录页面。
- 注册页面。
- 首页帖子列表。
- 帖子详情页。
- 发布帖子页。
- 评论展示和发布。
- 点赞和取消点赞接口接入。
- 用户个人中心。
- 我的帖子列表。
- 我的评论列表。
- 删除自己的帖子。
- 删除自己的评论。
- 设置页支持修改密码和退出登录。

### 文档

- API 文档位于 `docs/API_REFERENCE.md`。
- 已有前端架构、UI 规划等设计文档。
- 当前文档用于记录最新 MVP 状态。
