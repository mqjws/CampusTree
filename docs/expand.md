# 第一层：课程设计加分项（优先实现）

这些功能实现难度很低。

收益极高。

------

## 1 匿名等级系统 ⭐⭐⭐⭐⭐

类似贴吧等级。

用户：

```
发布帖子 +1经验

评论 +1经验

点赞获得 +2经验
```

等级：

```
Lv1 新生

Lv2 大一

Lv3 大二

Lv4 大三

Lv5 大四

Lv6 研究生
```

数据库：

```
user.exp
user.level
```

即可。

------

## 2 热门帖子排行 ⭐⭐⭐⭐⭐

首页：

```
今日热门

本周热门

总榜
```

排序依据：

```
点赞数
+
评论数
```

展示效果很好。

答辩老师很喜欢这种功能。

# 6 敏感词审核 ⭐⭐⭐⭐⭐

非常适合作为亮点。

例如：

```
赌博

诈骗

违法
```

数据库：

```
sensitive_word
```

发布帖子时：

```
if(content.contains(word))
{
    return error;
}
```

即可。
